# ohmyself! v1 — PRD

## Problem
Everything about a person is scattered. I want one place that holds it all as
markdown, that an agent can read and write, with privacy levels.

## Goals
- Store the brain as `.md` files (not a typical DB).
- Expose it over MCP (personal Claude) and REST (web/iOS).
- Per-note privacy: public / private / secret.
- Multi-tenant from day one.

## Non-goals (v1)
- Semantic/vector search (phase 2).
- Mobile app (later; REST is ready for it).

## Success
- I use my personal Claude over MCP daily.
- A public agent can answer about me using only public notes.
