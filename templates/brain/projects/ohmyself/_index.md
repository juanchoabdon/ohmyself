# ohmyself!

A second brain as loose markdown, exposed over MCP + REST, multi-tenant and
privacy-aware. The endgame: a public agent on my site that can answer about me
(within privacy), and a personal agent that knows everything and helps me decide.

- **Status:** building v1.
- **Stack:** TypeScript (core + MCP + Hono API), Next.js web, Supabase.
- See the PRD: [v1](projects/ohmyself/prds/v1.md).
